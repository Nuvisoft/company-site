# company-site

Corporate website built with **React + TypeScript + Vite**.

## Requirements

- Node.js ≥ 20.19
- Yarn ≥ 1.22

## Installation

```bash
yarn install
```

## Development

```bash
# Start the development server (with HMR)
yarn dev
# Default: http://localhost:5173
```

## Production build

```bash
# Create the /dist folder
yarn build

# Preview the production build locally
yarn preview
```
